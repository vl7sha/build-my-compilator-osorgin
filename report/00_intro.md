## Введение

Цель курсовой работы — построить основу компилятора для учебного языка **BNLang**: выполнить лексический и синтаксический разбор входной программы и формировать диагностические сообщения об ошибках.

В качестве инструмента генерации анализаторов используется **ANTLR 4**. Формальное описание языка задано грамматикой `BNLang.g4`, составленной по спецификации из `БН.md` (вариант 58 / 341111). Для проверки используются примеры из каталога `examples/`.

На текущем этапе проект ориентирован на режим **«только ошибки»**: результатом работы является список диагностик (лексических/синтаксических), без построения промежуточного кода и без выполнения программы.
