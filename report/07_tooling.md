## 7 Работа с программным средством

Проект использует ANTLR 4 для генерации лексера/парсера из грамматики `BNLang.g4`.

### 7.1 Подготовка

1) Поместить ANTLR jar в каталог `tools/` (например `tools/antlr-4.13.2-complete.jar`).

2) Сгенерировать Java‑исходники лексера/парсера:

```bash
./scripts/gen-java.sh
```

Сгенерированные файлы появятся в `gen/java/`.

### 7.2 Проверка файла (режим «только ошибки»)

Для проверки входного файла с выводом только диагностик (без дерева разбора):

```bash
./scripts/check-java.sh examples/01_numbers.bn
```

Если во входной программе есть ошибки, ANTLR выведет сообщения с позициями.

### 7.3 Примеры

Набор примеров расположен в каталоге `examples/`. Можно подставлять любой файл `.bn`:

```bash
./scripts/check-java.sh examples/03_if_else.bn
```
